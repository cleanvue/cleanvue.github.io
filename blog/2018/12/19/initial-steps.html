Initial Steps

I'm going to create a new project in the my local computer and will save the list of packages that the project uses.

Let's create a new project using vue-cli. To create a new project, type vue create jaffa into the command line. This command will ask you for different set options. Answer accordingly and select the things required. My configuration is as follows: Default Preset: (Babel, ESLint)

This is the bare default configuration for a new vue.js application created using the vue-cli. The command takes a little time to fetch all the dependencies and link them with the project. My internet is pretty slow. So, It takes around 5 minutes for me to complete the setup.

After the process got completed, I've actually ran yarn list and saved the output to deps.txt in this repo. Each time we do some progress, I'll run yarn again and then yarn list and save it to deps.txt. By this way, we can easily identify our progress.
